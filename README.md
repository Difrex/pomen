# pomen

Simple console menu

Run ```./pomen.pl makecache``` first

# Example

```
[work pomen]$ ./pomen.pl /
/Application
/AudioVideo
/Development
/GNOME
/GTK
/Game
/Graphics
/Network
/Office
/Qt
/Settings
/System
/X-Fedora
/X-Red-Hat-Base
[work pomen]$ ./pomen.pl /Network
/Network
    /Email/
    /FileTransfer/
    /InstantMessaging/
    /Telephony/
    /WebBrowser/
[work pomen]$ ./pomen.pl /Network/InstantMessaging/

    InstantMessaging/
        Клиент обмена мгновенными сообщениями Pidgin    pidgin
```
